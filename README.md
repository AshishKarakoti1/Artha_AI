# 🚀 ArthaAI

### AI-Powered Personal Finance & Goal Planning System (India-Focused)

---

## 📌 Project Overview

**ArthaAI** is an intelligent personal finance advisory system designed to help users **track expenses, plan financial goals, analyze savings behavior, and receive actionable recommendations**.

The system is built with a **modular, scalable architecture** and focuses on solving real-world financial challenges faced by **students and young professionals in India**.

> This project is an academic and copyrighted work developed at **Chitkara University, Punjab**. 

---

## 🎯 Problem Statement

Modern users struggle with:

* Fragmented financial tools
* Lack of unified financial planning systems
* Low financial literacy
* Complex Indian tax structures

Most existing apps only solve **isolated problems** (like expense tracking), but do not provide **holistic financial guidance**.

---

## 🎯 Objective

To build a system that:

* Tracks income, expenses, and savings
* Analyzes financial behavior
* Provides goal-based planning
* Generates intelligent financial recommendations

## 🧱 System Architecture

```text
Client Layer (React Frontend)
        ↓
API Layer (Express Backend)
        ↓
Business Logic Layer
        ↓
Recommendation Engine
        ↓
Database Layer (MongoDB)
```

The system follows a **modular layered architecture** ensuring scalability and maintainability. 

---

## 🧠 Core Logic

The recommendation engine uses **rule-based financial decision logic**, including:

* Savings ratio analysis
* Emergency fund evaluation
* Risk-based suggestions
* Goal feasibility checks

Example:

* Low savings → Suggest increase in discipline
* No emergency fund → Prioritize safety buffer

---

## 🗄️ Database Design

* **Users Collection** → Profile, income, risk
* **Expenses Collection** → Transactions & categories
* **Goals Collection** → Targets & deadlines
* **Investments Collection** → Financial assets

Uses **MongoDB document-based schema** for flexibility.

## 🚀 Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-repo/artha-ai.git
cd artha-ai
```

---

### 2. Backend Setup

```bash
cd server
npm install
```

Create `.env`:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

Run:

```bash
npm run dev
```

---

### 3. Frontend Setup

```bash
cd client
npm install
npm run dev
```
